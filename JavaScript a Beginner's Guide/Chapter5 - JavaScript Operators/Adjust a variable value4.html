The \n character is used to find a newline character in JavaScript.[1]

The "\n" character is used to find a newline character in JavaScript.[1]

Browsers doesn't treat \r\n as real new-lines.[2]

However, there is a workaround.
"Browser does treat it as new-lines if you specify the correct css style."[2][3][4]
